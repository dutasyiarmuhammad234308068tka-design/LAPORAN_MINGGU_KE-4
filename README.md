# LAPORAN_MINGGU_KE-4
Program ini dibuat untuk membangun sistem monitoring berbasis IoT dengan mengakses IP Camera, mendeteksi keberadaan tangan menggunakan MediaPipe, serta mengirimkan hasil deteksi secara real-time melalui MQTT. Tujuannya adalah mengintegrasikan computer vision dan komunikasi jaringan dalam satu sistem sederhana dan fungsional.
